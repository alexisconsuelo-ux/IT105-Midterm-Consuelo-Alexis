APPOINTMENT SYSTEM

SYSTEM DESCRIPTION:
This system is a databased system designed to organize and manage appointments effecienty.
The system allows both users to store an retive records/data.

TABLES DESCRIPTION:

-Clients: Stores clients information (Name, Email, contact number)

-Providers: Stores the providers details

-Appointments: Records appointments (Date, Time, Status)

-Payments: Stores payment details


FEATURES IMPLEMENTED:

-Implemented inserting, updating, and deleting records

-Retrieveing data (SELECT)

-Combining data (JOIN)

-Advanced data retrieval (SUBQUERY)

-Indexing queries
